# ✅ Вариант 16

## 🟢 Задание 1 (базовый уровень)

### Постановка задачи

Создайте пользовательскую форму **«Заявка на языковые курсы»** для последовательного ввода данных нескольких слушателей.

### Элементы формы

Добавьте:

1.  Текст (строка) — ФИО слушателя (`student_name`)
2.  Текст (строка) — Номер телефона (`phone_number`)
3.  Число — Желаемое количество часов в неделю (`hours_per_week`, min = 2, max = 20)
4.  Кнопки:

   *   Подтвердить (ОК)
   *   Отмена

---

### Подготовка данных

С помощью любого ИИ-сервиса (DeepSeek / ChatGPT) сгенерируйте **10 тестовых записей** вида:

```
ФИО, Телефон, Часы_в_неделю
```

---

### Требования к алгоритму

1.  Форма вызывается **в цикле 10 раз**.
2.  На каждой итерации вводится один набор данных.
3.  После нажатия ОК данные добавляются в файл `language_courses.txt`.
4.  Каждая запись сохраняется с новой строки в формате:

   ```
   ФИО;Телефон;Часы
   ```
5.  При нажатии Отмена ввод прекращается.
6.  После завершения вывести сообщение:

   ```
   Оформлено заявок: N
   ```

# 🔵 Задание 2 (повышенный уровень)

### Постановка задачи

Создайте форму **«Расширенная заявка на языковые курсы»** с дополнительными элементами, условиями показа и использованием кнопки типа **Триггер**.

---

### Элементы формы

1.  Текст — ФИО слушателя (`student_name`)
2.  Текст — Номер телефона (`phone_number`)
3.  Число — Возраст слушателя (`student_age`)
4.  Чекбокс — Необходим учебник (`needs_textbook`)
5.  Текст — Артикул учебника (`textbook_sku`)

   *   отображать только если `needs_textbook = Истина`
6.  Селект — Изучаемый язык (`target_language`)

   *   варианты: english, german, spanish
7.  Кнопки:

   *   Триггер **«Проверить заявку»**
   *   ОК
   *   Отмена

---

### Логика кнопки Триггер

Создайте функцию `validate_language_request(form_result)`, которая:

1.  Проверяет:

   *   `phone_number` содержит цифру `7`
   *   И (`student_age` >= 14 ИЛИ `needs_textbook` = Истина)
2.  Формирует текст:

   *   «Заявка одобрена»
   *   или «Ошибка: неверный телефон или возраст без учебника»

Результатом работы функции является вывод сообщения на экран

⚠ Форма не должна закрываться при нажатии Триггера.

---

### Динамические параметры при вызове формы

Программно задать:

*   заголовок окна: **Языковая школа 🎓**
*   изменить цвет фона;
*   значение по умолчанию: `target_language = english`.

---

### Подготовка данных

Сгенерируйте с помощью ИИ **10 расширенных записей**:

```
ФИО, Телефон, Возраст, Учебник (да/нет), Артикул_учебника, Язык
```

---

### Требования к алгоритму

1.  Ввод выполняется последовательно для 10 слушателей.
2.  Перед подтверждением обязательно нажимать кнопку «Проверить заявку».
3.  После нажатия ОК данные сохраняются в файл `language_courses_extended.csv`:

```
ФИО;Телефон;Возраст;Учебник;Артикул_учебника;Язык
```

4.  Если форма возвращает пустое значение (Отмена), выполнение прекращается.
