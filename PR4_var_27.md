# ✅ Вариант 27

## 🟢 Задание 1 (базовый уровень)

### Постановка задачи

Создайте пользовательскую форму **«Регистрация клиента туристического агентства»** для последовательного ввода данных нескольких туристов, желающих оформить путёвку.

### Элементы формы

Добавьте:

1.  Текст (строка) — ФИО туриста (`tourist_fullname`)
2.  Текст (строка) — Номер загранпаспорта (`passport_intl`)
3.  Число — Количество дней тура (`tour_days`, min = 3, max = 30)
4.  Кнопки:

   *   Подтвердить (ОК)
   *   Отмена

---

### Подготовка данных

С помощью любого ИИ-сервиса (DeepSeek / ChatGPT) сгенерируйте **10 тестовых записей** вида:

```
ФИО, Номер_загранпаспорта, Дней_тура
```

---

### Требования к алгоритму

1.  Форма вызывается **в цикле 10 раз**.
2.  На каждой итерации вводится один набор данных.
3.  После нажатия ОК данные добавляются в файл `travel_clients.txt`.
4.  Каждая запись сохраняется с новой строки в формате:

   ```
   ФИО;Номер_загранпаспорта;Дней_тура
   ```
5.  При нажатии Отмена ввод прекращается.
6.  После завершения вывести сообщение:

   ```
   Зарегистрировано туристов: N
   ```

# 🔵 Задание 2 (повышенный уровень)

### Постановка задачи

Создайте форму **«Расширенная регистрация тура»** с дополнительными элементами, условиями показа и использованием кнопки типа **Триггер**.

---

### Элементы формы

1.  Текст — ФИО туриста (`tourist_fullname`)
2.  Текст — Номер загранпаспорта (`passport_intl`)
3.  Число — Количество дней тура (`tour_days`)
4.  Чекбокс — Нужна визовая поддержка (`needs_visa_support`)
5.  Текст — Страна назначения для визы (`visa_country`)

   *   отображать только если `needs_visa_support = Истина`
6.  Селект — Тип размещения (`accommodation_type`)

   *   варианты: hotel, apartment, resort, hostel
7.  Кнопки:

   *   Триггер **«Проверить тур»**
   *   ОК
   *   Отмена

---

### Логика кнопки Триггер

Создайте функцию `validate_tour_booking(form_result)`, которая:

1.  Проверяет:

   *   (Длина номера загранпаспорта == 9 И номер состоит только из цифр)
   *   И (Количество дней >= 5 ИЛИ (Нужна визовая поддержка = Истина И длина страны назначения >= 4))
2.  Формирует текст:

   *   «Тур подтверждён»
   *   или «Ошибка: неверный паспорт или недостаточно дней/данных для визы»

Результатом работы функции является вывод сообщения на экран.

⚠ Форма не должна закрываться при нажатии Триггера.

---

### Динамические параметры при вызове формы

Программно задать:

*   заголовок окна: **Travel Agency ✈️**
*   изменить цвет фона;
*   значение по умолчанию: `accommodation_type = hotel`.

---

### Подготовка данных

Сгенерируйте с помощью ИИ **10 расширенных записей**:

```
ФИО, Номер_загранпаспорта, Дней_тура, Виза_поддержка (да/нет), Страна_визы, Тип_размещения
```

---

### Требования к алгоритму

1.  Ввод выполняется последовательно для 10 туристов.
2.  Перед подтверждением обязательно нажимать кнопку «Проверить тур».
3.  После нажатия ОК данные сохраняются в файл `travel_clients_extended.csv`:

```
ФИО;Номер_загранпаспорта;Дней_тура;Виза_поддержка;Страна_визы;Тип_размещения
```

4.  Если форма возвращает пустое значение (Отмена), выполнение прекращается.
