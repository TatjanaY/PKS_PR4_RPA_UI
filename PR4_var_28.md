# ✅ Вариант 28

## 🟢 Задание 1 (базовый уровень)

### Постановка задачи

Создайте пользовательскую форму **«Заявка на аренду оборудования»** для последовательного ввода данных нескольких обращений клиентов.

### Элементы формы

Добавьте:

1.  Текст (строка) — ФИО арендатора (`renter_name`)
2.  Текст (строка) — Email для договора (`contract_email`)
3.  Число — Срок аренды в днях (`rental_days`, min = 1, max = 90)
4.  Кнопки:

   *   Подтвердить (ОК)
   *   Отмена

---

### Подготовка данных

С помощью любого ИИ-сервиса (DeepSeek / ChatGPT) сгенерируйте **10 тестовых записей** вида:

```
ФИО, Email, Срок_аренды
```

---

### Требования к алгоритму

1.  Форма вызывается **в цикле 10 раз**.
2.  На каждой итерации вводится один набор данных.
3.  После нажатия ОК данные добавляются в файл `equipment_rentals.txt`.
4.  Каждая запись сохраняется с новой строки в формате:

   ```
   ФИО;Email;Срок
   ```
5.  При нажатии Отмена ввод прекращается.
6.  После завершения вывести сообщение:

   ```
   Оформлено заявок на аренду: N
   ```

# 🔵 Задание 2 (повышенный уровень)

### Постановка задачи

Создайте форму **«Расширенная заявка на аренду оборудования»** с дополнительными элементами, условиями показа и использованием кнопки типа **Триггер**.

---

### Элементы формы

1.  Текст — ФИО арендатора (`renter_name`)
2.  Текст — Email для договора (`contract_email`)
3.  Число — Срок аренды в днях (`rental_days`)
4.  Чекбокс — Требуется доставка (`needs_delivery`)
5.  Текст — Адрес доставки (`delivery_address`)

   *   отображать только если `needs_delivery = Истина`
6.  Селект — Категория оборудования (`equipment_category`)

   *   варианты: audio, video, lighting
7.  Кнопки:

   *   Триггер **«Проверить аренду»**
   *   ОК
   *   Отмена

---

### Логика кнопки Триггер

Создайте функцию `validate_equipment_rental(form_result)`, которая:

1.  Проверяет:

   *   `contract_email` содержит символ `@` И длина строки >= 10
   *   И (`rental_days` <= 30 ИЛИ `needs_delivery` = Ложь)
2.  Формирует текст:

   *   «Аренда одобрена»
   *   или «Ошибка: неверный email или срок аренды превышает лимит для доставки»

Результатом работы функции является вывод сообщения на экран

⚠ Форма не должна закрываться при нажатии Триггера.

---

### Динамические параметры при вызове формы

Программно задать:

*   заголовок окна: **Аренда ProGear 🎬**
*   изменить цвет фона;
*   значение по умолчанию: `equipment_category = video`.

---

### Подготовка данных

Сгенерируйте с помощью ИИ **10 расширенных записей**:

```
ФИО, Email, Срок_аренды, Доставка (да/нет), Адрес_доставки, Категория
```

---

### Требования к алгоритму

1.  Ввод выполняется последовательно для 10 заявок.
2.  Перед подтверждением обязательно нажимать кнопку «Проверить аренду».
3.  После нажатия ОК данные сохраняются в файл `equipment_rentals_extended.csv`:

```
ФИО;Email;Срок_аренды;Доставка;Адрес_доставки;Категория
```

4.  Если форма возвращает пустое значение (Отмена), выполнение прекращается.
