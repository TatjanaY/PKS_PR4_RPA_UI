# ✅ Вариант 3

## 🟢 Задание 1 (базовый уровень)

### Постановка задачи

Создайте пользовательскую форму **«Регистрация игрока»** для последовательного ввода данных нескольких участников игровой платформы.

### Элементы формы

Добавьте:

1.  Текст (строка) — Никнейм (`nickname`)
2.  Текст (строка) — Логин (`login`)
3.  Число — Уровень доступа (`level`, min = 1, max = 50)
4.  Кнопки:

   *   Подтвердить (ОК)
   *   Отмена

---

### Подготовка данных

С помощью любого ИИ-сервиса (DeepSeek / ChatGPT) сгенерируйте **10 тестовых записей** вида:

```
Никнейм, Логин, Уровень
```

---

### Требования к алгоритму

1.  Форма вызывается **в цикле 10 раз**.
2.  На каждой итерации вводится один набор данных.
3.  После нажатия ОК данные добавляются в файл `gamers.txt`.
4.  Каждая запись сохраняется с новой строки в формате:

   ```
   Никнейм;Логин;Уровень
   ```
5.  При нажатии Отмена ввод прекращается.
6.  После завершения вывести сообщение:

   ```
   Зарегистрировано игроков: N
   ```

# 🔵 Задание 2 (повышенный уровень)

### Постановка задачи

Создайте форму **«Расширенная регистрация игрока»** с дополнительными элементами, условиями показа и использованием кнопки типа **Триггер**.

---

### Элементы формы

1.  Текст — Никнейм (`nickname`)
2.  Текст — Логин (`login`)
3.  Число — Уровень доступа (`level`)
4.  Чекбокс — Премиум аккаунт (`is_premium`)
5.  Текст — ID промокода (`promo_code`)

   *   отображать только если `is_premium = Истина`
6.  Селект — Предпочитаемый жанр (`genre`)

   *   варианты: rpg, shooter, strategy
7.  Кнопки:

   *   Триггер **«Активировать профиль»**
   *   ОК
   *   Отмена

---

### Логика кнопки Триггер

Создайте функцию `activate_profile(form_result)`, которая:

1.  Проверяет:

   *   (Уровень >= 10 И длина Никнейма >= 4 символов)
   *   ИЛИ (Премиум аккаунт = Истина)
2.  Формирует текст:

   *   «Профиль активирован»
   *   или «Ошибка: низкий уровень или короткий ник без премиума»

Результатом работы функции является вывод сообщения на экран

⚠ Форма не должна закрываться при нажатии Триггера.

---

### Динамические параметры при вызове формы

Программно задать:

*   заголовок окна: **Game Zone 🎮**
*   изменить цвет фона;
*   значение по умолчанию: `genre = rpg`.

---

### Подготовка данных

Сгенерируйте с помощью ИИ **10 расширенных записей**:

```
Никнейм, Логин, Уровень, Премиум (да/нет), Промокод, Жанр
```

---

### Требования к алгоритму

1.  Ввод выполняется последовательно для 10 игроков.
2.  Перед подтверждением обязательно нажимать кнопку «Активировать профиль».
3.  После нажатия ОК данные сохраняются в файл `gamers_extended.csv`:

```
Никнейм;Логин;Уровень;Премиум;Промокод;Жанр
```

4.  Если форма возвращает пустое значение (Отмена), выполнение прекращается.
