# ✅ Вариант 35

## 🟢 Задание 1 (базовый уровень)

### Постановка задачи

Создайте пользовательскую форму **«Регистрация участника онлайн-арт-школы»** для последовательного ввода данных нескольких учеников, желающих записаться на мастер-класс по рисованию, живописи или дизайну.

### Элементы формы

Добавьте:

1.  Текст (строка) — ФИО ученика (`student_fullname`)
2.  Текст (строка) — Никнейм в школе (`art_nickname`)
3.  Число — Уровень подготовки (от 1 до 10) (`skill_level`, min = 1, max = 10)
4.  Кнопки:

   *   Подтвердить (ОК)
   *   Отмена

---

### Подготовка данных

С помощью любого ИИ-сервиса (DeepSeek / ChatGPT) сгенерируйте **10 тестовых записей** вида:

```
ФИО, Никнейм, Уровень_подготовки
```

---

### Требования к алгоритму

1.  Форма вызывается **в цикле 10 раз**.
2.  На каждой итерации вводится один набор данных.
3.  После нажатия ОК данные добавляются в файл `art_school_students.txt`.
4.  Каждая запись сохраняется с новой строки в формате:

   ```
   ФИО;Никнейм;Уровень_подготовки
   ```
5.  При нажатии Отмена ввод прекращается.
6.  После завершения вывести сообщение:

   ```
   Зарегистрировано учеников: N
   ```

# 🔵 Задание 2 (повышенный уровень)

### Постановка задачи

Создайте форму **«Расширенная регистрация в арт-школу»** с дополнительными элементами, условиями показа и использованием кнопки типа **Триггер**.

---

### Элементы формы

1.  Текст — ФИО ученика (`student_fullname`)
2.  Текст — Никнейм в школе (`art_nickname`)
3.  Число — Уровень подготовки (`skill_level`)
4.  Чекбокс — Есть свой графический планшет (`has_graphics_tablet`)
5.  Текст — Модель планшета (`tablet_model`)

   *   отображать только если `has_graphics_tablet = Истина`
6.  Селект — Направление мастер-класса (`art_direction`)

   *   варианты: drawing, painting, design
7.  Кнопки:

   *   Триггер **«Проверить зачисление»**
   *   ОК
   *   Отмена

---

### Логика кнопки Триггер

Создайте функцию `validate_art_enrollment(form_result)`, которая:

1.  Проверяет:

   *   (Длина никнейма >= 4 И никнейм не содержит спецсимволов кроме `_`)
   *   И (Уровень подготовки >= 3 ИЛИ (Направление == "design" И Есть планшет = Истина))
2.  Формирует текст:

   *   «Ученик зачислен на курс»
   *   или «Ошибка: короткий никнейм или уровень/оборудование не соответствуют направлению»

Результатом работы функции является вывод сообщения на экран

⚠ Форма не должна закрываться при нажатии Триггера.

---

### Динамические параметры при вызове формы

Программно задать:

*   заголовок окна: **Art School Online 🎨**
*   изменить цвет фона;
*   значение по умолчанию: `art_direction = drawing`.

---

### Подготовка данных

Сгенерируйте с помощью ИИ **10 расширенных записей**:

```
ФИО, Никнейм, Уровень_подготовки, Планшет (да/нет), Модель_планшета, Направление
```

---

### Требования к алгоритму

1.  Ввод выполняется последовательно для 10 учеников.
2.  Перед подтверждением обязательно нажимать кнопку «Проверить зачисление».
3.  После нажатия ОК данные сохраняются в файл `art_school_students_extended.csv`:

```
ФИО;Никнейм;Уровень_подготовки;Планшет;Модель_планшета;Направление
```

4.  Если форма возвращает пустое значение (Отмена), выполнение прекращается.
