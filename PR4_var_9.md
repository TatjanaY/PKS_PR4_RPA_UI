# ✅ Вариант 9

## 🟢 Задание 1 (базовый уровень)

### Постановка задачи

Создайте пользовательскую форму **«Регистрация подписчика онлайн-сервиса»** для последовательного ввода данных нескольких пользователей стриминговой платформы.

### Элементы формы

Добавьте:

1.  Текст (строка) — Имя пользователя (`username`)
2.  Текст (строка) — Адрес почты (`user_email`)
3.  Число — Длительность подписки в месяцах (`sub_duration`, min = 1, max = 24)
4.  Кнопки:

   *   Подтвердить (ОК)
   *   Отмена

---

### Подготовка данных

С помощью любого ИИ-сервиса (DeepSeek / ChatGPT) сгенерируйте **10 тестовых записей** вида:

```
Имя_пользователя, Адрес_почты, Длительность_мес
```

---

### Требования к алгоритму

1.  Форма вызывается **в цикле 10 раз**.
2.  На каждой итерации вводится один набор данных.
3.  После нажатия ОК данные добавляются в файл `subscribers.txt`.
4.  Каждая запись сохраняется с новой строки в формате:

   ```
   Имя_пользователя;Адрес_почты;Длительность_мес
   ```
5.  При нажатии Отмена ввод прекращается.
6.  После завершения вывести сообщение:

   ```
   Зарегистрировано подписчиков: N
   ```

# 🔵 Задание 2 (повышенный уровень)

### Постановка задачи

Создайте форму **«Расширенная регистрация подписчика»** с дополнительными элементами, условиями показа и использованием кнопки типа **Триггер**.

---

### Элементы формы

1.  Текст — Имя пользователя (`username`)
2.  Текст — Адрес почты (`user_email`)
3.  Число — Длительность подписки в месяцах (`sub_duration`)
4.  Чекбокс — Автопродление подписки (`auto_renew`)
5.  Текст — Промокод (`promo_input`)

   *   отображать только если `auto_renew = Истина`
6.  Селект — Качество стриминга (`stream_quality`)

   *   варианты: sd, hd, 4k
7.  Кнопки:

   *   Триггер **«Валидировать подписку»**
   *   ОК
   *   Отмена

---

### Логика кнопки Триггер

Создайте функцию `validate_subscription(form_result)`, которая:

1.  Проверяет:

   *   (Длина имени пользователя >= 5 И имя не содержит пробелов)
   *   ИЛИ (Автопродление = Истина И длина Промокода == 8)
2.  Формирует текст:

   *   «Подписка активирована»
   *   или «Ошибка: короткое имя или неверный промокод при автопродлении»

Результатом работы функции является вывод сообщения на экран

⚠ Форма не должна закрываться при нажатии Триггера.

---

### Динамические параметры при вызове формы

Программно задать:

*   заголовок окна: **Stream Service 🎬**
*   изменить цвет фона;
*   значение по умолчанию: `stream_quality = hd`.

---

### Подготовка данных

Сгенерируйте с помощью ИИ **10 расширенных записей**:

```
Имя_пользователя, Адрес_почты, Длительность_мес, Автопродление (да/нет), Промокод, Качество
```

---

### Требования к алгоритму

1.  Ввод выполняется последовательно для 10 подписчиков.
2.  Перед подтверждением обязательно нажимать кнопку «Валидировать подписку».
3.  После нажатия ОК данные сохраняются в файл `subscribers_extended.csv`:

```
Имя_пользователя;Адрес_почты;Длительность_мес;Автопродление;Промокод;Качество
```

4.  Если форма возвращает пустое значение (Отмена), выполнение прекращается.
