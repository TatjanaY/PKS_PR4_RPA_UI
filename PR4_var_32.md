# ✅ Вариант 32

## 🟢 Задание 1 (базовый уровень)

### Постановка задачи

Создайте пользовательскую форму **«Заявка на участие в конкурсе»** для последовательного ввода данных нескольких участников.

### Элементы формы

Добавьте:

1.  Текст (строка) — ФИО участника (`contestant_name`)
2.  Текст (строка) — Email для связи (`contest_email`)
3.  Число — Возраст участника (`contestant_age`, min = 12, max = 65)
4.  Кнопки:

   *   Подтвердить (ОК)
   *   Отмена

---

### Подготовка данных

С помощью любого ИИ-сервиса (DeepSeek / ChatGPT) сгенерируйте **10 тестовых записей** вида:

```
ФИО, Email, Возраст
```

---

### Требования к алгоритму

1.  Форма вызывается **в цикле 10 раз**.
2.  На каждой итерации вводится один набор данных.
3.  После нажатия ОК данные добавляются в файл `contest_applications.txt`.
4.  Каждая запись сохраняется с новой строки в формате:

   ```
   ФИО;Email;Возраст
   ```
5.  При нажатии Отмена ввод прекращается.
6.  После завершения вывести сообщение:

   ```
   Принято заявок на конкурс: N
   ```

# 🔵 Задание 2 (повышенный уровень)

### Постановка задачи

Создайте форму **«Расширенная заявка на конкурс»** с дополнительными элементами, условиями показа и использованием кнопки типа **Триггер**.

---

### Элементы формы

1.  Текст — ФИО участника (`contestant_name`)
2.  Текст — Email для связи (`contest_email`)
3.  Число — Возраст участника (`contestant_age`)
4.  Чекбокс — Есть портфолио (`has_portfolio`)
5.  Текст — Ссылка на портфолио (`portfolio_link`)

   *   отображать только если `has_portfolio = Истина`
6.  Селект — Номинация конкурса (`contest_nomination`)

   *   варианты: art, tech, literature
7.  Кнопки:

   *   Триггер **«Проверить заявку»**
   *   ОК
   *   Отмена

---

### Логика кнопки Триггер

Создайте функцию `validate_contest_application(form_result)`, которая:

1.  Проверяет:

   *   `contest_email` содержит символ `@` И не заканчивается на `test`
   *   И (`contestant_age` >= 16 ИЛИ `has_portfolio` = Истина)
2.  Формирует текст:

   *   «Заявка соответствует требованиям конкурса»
   *   или «Ошибка: неверный email или возраст без портфолио»

Результатом работы функции является вывод сообщения на экран

⚠ Форма не должна закрываться при нажатии Триггера.

---

### Динамические параметры при вызове формы

Программно задать:

*   заголовок окна: **Конкурс Talent🏆**
*   изменить цвет фона;
*   значение по умолчанию: `contest_nomination = art`.

---

### Подготовка данных

Сгенерируйте с помощью ИИ **10 расширенных записей**:

```
ФИО, Email, Возраст, Портфолио (да/нет), Ссылка_портфолио, Номинация
```

---

### Требования к алгоритму

1.  Ввод выполняется последовательно для 10 участников.
2.  Перед подтверждением обязательно нажимать кнопку «Проверить заявку».
3.  После нажатия ОК данные сохраняются в файл `contest_applications_extended.csv`:

```
ФИО;Email;Возраст;Портфолио;Ссылка_портфолио;Номинация
```

4.  Если форма возвращает пустое значение (Отмена), выполнение прекращается.
