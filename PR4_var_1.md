# ✅ Вариант 1

## 🟢 Задание 1 (базовый уровень)

### Постановка задачи

Создайте пользовательскую форму **«Анкета клиента фитнес-клуба»** для последовательного ввода данных нескольких посетителей.

### Элементы формы

Добавьте:

1.  Текст (строка) — ФИО (`full_name`)
2.  Текст (строка) — Телефон (`phone`)
3.  Число — Количество посещений в неделю (`visits_per_week`, min = 1, max = 7)
4.  Кнопки:

   *   Подтвердить (ОК)
   *   Отмена

---

### Подготовка данных

С помощью любого ИИ-сервиса (DeepSeek / ChatGPT) сгенерируйте **10 тестовых записей** вида:

```
ФИО, Телефон, Посещения_в_неделю
```

---

### Требования к алгоритму

1.  Форма вызывается **в цикле 10 раз**.
2.  На каждой итерации вводится один набор данных.
3.  После нажатия ОК данные добавляются в файл `fitness_clients.txt`.
4.  Каждая запись сохраняется с новой строки в формате:

   ```
   ФИО;Телефон;Посещения
   ```
5.  При нажатии Отмена ввод прекращается.
6.  После завершения вывести сообщение:

   ```
   Зарегистрировано клиентов: N
   ```

# 🔵 Задание 2 (повышенный уровень)

### Постановка задачи

Создайте форму **«Расширенная анкета фитнес-клуба»** с дополнительными элементами, условиями показа и использованием кнопки типа **Триггер**.

---

### Элементы формы

1.  Текст — ФИО (`full_name`)
2.  Текст — Телефон (`phone`)
3.  Число — Год рождения (`birth_year`)
4.  Чекбокс — Нужен персональный тренер (`needs_trainer`)
5.  Текст — ID тренера (`trainer_id`)

   *   отображать только если `needs_trainer = Истина`
6.  Селект — Зона доступа (`access_zone`)

   *   варианты: cardio, strength, mix
7.  Кнопки:

   *   Триггер **«Проверить доступ»**
   *   ОК
   *   Отмена

---

### Логика кнопки Триггер

Создайте функцию `validate_access(form_result)`, которая:

1.  Проверяет:

   *   телефон начинается с символа `8` или `+`
   *   ИЛИ (год рождения <= 2005 И нужен тренер = Истина)
2.  Формирует текст:

   *   «Доступ разрешён»
   *   или «Ошибка: неверный телефон или возраст без тренера»

Результатом работы функции является вывод сообщения на экран

⚠ Форма не должна закрываться при нажатии Триггера.

---

### Динамические параметры при вызове формы

Программно задать:

*   заголовок окна: **Фитнес Клуб 🏋️**
*   изменить цвет фона;
*   значение по умолчанию: `access_zone = mix`.

---

### Подготовка данных

Сгенерируйте с помощью ИИ **10 расширенных записей**:

```
ФИО, Телефон, Год_рождения, Тренер (да/нет), ID_тренера, Зона
```

---

### Требования к алгоритму

1.  Ввод выполняется последовательно для 10 клиентов.
2.  Перед подтверждением обязательно нажимать кнопку «Проверить доступ».
3.  После нажатия ОК данные сохраняются в файл `fitness_clients_extended.csv`:

```
ФИО;Телефон;Год_рождения;Тренер;ID_тренера;Зона
```

4.  Если форма возвращает пустое значение (Отмена), выполнение прекращается.
